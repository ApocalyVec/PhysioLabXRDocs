Run from source¶

Note

This App is still in alpha development build. Please follow this guide below to learn how to build and run the App.

RealityNavigation App requires python3.8+. Clone the project from RealityNavigation’s repo page.

Navigation to the root folder of the downloaded repo, install all the prerequisites with the following command:

pip install -r requirements.txt

Run main.py to start the app:

python main.py
