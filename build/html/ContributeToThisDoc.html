Contribute to this Doc¶

If you would like to add contents to this doc, you can follow these instructions on how to make changes and compile them

1.Fork the the RealityNavigationDocs repo. 2.Clone the forked repo to your local directory, with command like this:

git clone https://github.com/<GitHubUserName>/RealityNavigationDocs

3.Go to <PathToTheClonedRepo>/source, you can edit existing pages here or add new pages. You could include a new page in “Further Information” in index.rst for easier navigation.

4.Install the package need for compiling the webpage

pip install sphinx

5.In the root directory of the project, compile the html using the command

make html

6.View your changes by openning <PathToTheClonedRepo>/build/html/index.html

7.Submit your change by creating a pull request in GitHub. Read here on how to create pull request from fork.
